# 简介
&ensp;&ensp;无侵入性的上下留白和左右留白

# ✨ 特性
- 支持 vue(2.x)
- 支持国际化
- 支持修改主题
- 支持动态引入(babel-plugin-import)

# 🖥 兼容环境
- 现代浏览器，IE11

# 📦 安装
```javascript
npm install @baifendian/adherev-ui-space --save
``` 

```javascript
yarn add @baifendian/adherev-ui-space
```
